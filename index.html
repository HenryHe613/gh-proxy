GitHub 文件加速

1. GitHub文件链接带不带协议头都可以，支持release、archive以及文件，右键复制出来的链接都是符合标准的。

2. release、archive使用cf加速，文件会跳转至JsDelivr

3. 注意，不支持项目文件夹

1. 分支源码：https://github.com/HenryHe613/blog/archive/master.zip

2. release源码：https://github.com/HenryHe613/blog/archive/v0.1.0.tar.gz

3. release文件：https://github.com/HenryHe613/blog/releases/download/v0.1.0/example.zip

4. 分支文件：https://github.com/HenryHe613/blog/blob/master/filename

项目基于Cloudflare Workers，开源于GitHub 南岩のBlog
